# SlyxUp React App

This project was bootstrapped with [SlyxUp CLI](https://github.com/slyxup/cli).

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 What's Included

- ⚡️ **Vite** - Lightning fast build tool
- ⚛️ **React 18** - Latest React with Hooks
- 🔷 **TypeScript** - Type safety out of the box
- 🎨 **CSS** - Ready for styling
- 📏 **ESLint** - Code linting configured

## 🎨 Adding Features

Add features to your project with SlyxUp:

```bash
# Add Tailwind CSS
slyxup add tailwind

# Add shadcn/ui components
slyxup add shadcn

# Add Lucide icons
slyxup add lucide
```
