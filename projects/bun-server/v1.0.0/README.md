# Bun Server

A fast HTTP server using Bun runtime and APIs.

## Usage

```bash
bun install
bun run dev
```

## Scripts

- `bun run dev` - Start development server with hot reload
- `bun start` - Run production server

## Features

- Bun runtime - Incredibly fast JavaScript runtime
- Native TypeScript support
- Built-in hot reload
- Fast startup and execution
- Native HTTP server using Bun.serve()
