# Fastify API

A fast and low overhead Fastify v4 + TypeScript starter.

## Usage

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Run production server

## Features

- Fastify v4 - Fast and low overhead web framework
- TypeScript support
- CORS enabled
- Health check endpoint
- Route organization structure
