# GraphQL API

A GraphQL server with Apollo Server v4 + TypeScript.

## Usage

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Run production server

## Features

- Apollo Server v4
- GraphQL API
- TypeScript support
- Express integration
- GraphQL Playground available at `/graphql`
